# Restaurant QR Ordering SaaS

Modern restaurant QR ordering SaaS built with Next.js 14 App Router, TypeScript, Tailwind CSS, Zustand, and localStorage-backed mock data.

## Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- Zustand
- Frontend-only localStorage state

## Setup

1. Install dependencies:

```bash
npm install
```

2. Ensure PostgreSQL is running and update `.env` if needed.
2. Start the app:

```bash
npm run dev
```
